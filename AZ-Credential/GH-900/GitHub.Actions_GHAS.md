# 측정된 사용량 보고서
GitHub는 요금제 제품의 사용량을 추적하기 위한 자세한 청구 및 소비 보고서를 제공한다. 이는 보고서를 통해 관리자는 비용을 모니터링하고, 리소스를 효율적으로 할당하며 조직 정책을 준수할 수 있다.

## GitHub Actions Job Time
GitHub Actions는 가상 머신에서 워크플로가 실행되는 CI/CD 자동화 도구이다. 이러한 워크플로에서 소요되는 시간은 리포시토리 유형, 실행기 유형 및 사용량에 따라 측정된다.

### 소비 추적
* GitHub 조직 또는 계정에서 설정 -> 청구로 이동한다.
* GitHub Actions 섹션에서 사용된 시간을 확인할 수 있다.
* 사용량은 리포지토리, 실행기 유형 및 남은 할당량에 따라 세분화된다.
