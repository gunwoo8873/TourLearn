## Moile
장소와 환경에 제약없이 빠르게 작업을 수행할 수 있는 방법을 제공한다 신뢰할 수 있는 자체 클라이언트 애플리케이션을 통해 데이터에 액세스하는 안전한 방법이며 IOS와 Android 디바이스에 사용이 가능하다. 다음과 같이 여러 작업을 수행이 가능하다.
    
* 스레드나 기타 알림을 음소거 하거나 혹은 확인 및 필터링 하여 github.com 알림을 관리 혹은 심사 및 삭제가 가능하다.
* 이슈와 끌어오기 요청을 읽고 협업을 요청한다.
* 사용자, 리포지토리, 조직을 검색하고 찾아볼 수 있어 상호 작용을 가능
* 외부 사용자 이름을 언급할 때 알림을 수신
* 특정 사용자 지정 시간에 대한 푸시 알림 예약
* 2단계 인증을 사용하여 계정 보호
* 인식할 수 없는 디바이스에서 로그인 시도 확인

## Desktop
Git 워크플로를 간소화하는 데 도움이 되는 오픈 소스 애플리케이션이다. 사용자와 팀 간의 협업을 용이하게 하고 팀 내에서 Git 및 GitHub 모범 사례를 공유할 수 있으며 Windows와 MacOS에서 사용 가능하다. 다음과 같이 여러 수행 가능한 작업이 존재한다.

* 리포지토리를 추가하여 기존 프로젝트를 로컬로 관리
* GitHub에서 리포지토리를 복제하여 신속하게 로컬로 프로젝트를 설정하고 작업 시작
* 커밋에 변경 내용을 대화형으로 추가하면 커밋하기 전에 변경 내용을 보다 쉽게 관리하고 검토할 수 있다.
* 공동 작성자를 커밋에 빠르게 추가하여 공동 작업 프로젝트에서 적절한 특성을 보장한다.
* CI 상태를 확인하여 코드 품질을 확인하고 공동 작업을 간소화한다.
* 변경된 이미지를 비교하여 이미지 파일의 업데이트를 시각적으로 검토
